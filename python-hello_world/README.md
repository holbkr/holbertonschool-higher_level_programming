# Python - Hello, World

This project contains introductory Python scripts covering basic concepts like printing, strings, and variables.

## Tasks

0. **2-print.py** - Prints a specific string
1. **3-print_number.py** - Prints an integer with a string using f-strings
2. **4-print_float.py** - Prints a float with 2 decimal places
3. **5-print_string.py** - Prints a string 3 times and its first 9 characters
4. **6-concat.py** - Concatenates two strings
5. **7-edges.py** - Extracts parts of a string
6. **8-concat_edges.py** - Creates a new string from parts of an existing string
7. **9-easter_egg.py** - Prints the Zen of Python

## Requirements

- Python 3.8.*
- Ubuntu 20.04 LTS
- pycodestyle 2.7.*
